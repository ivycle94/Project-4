# Installation Instructions:
1. fork & clone this repo
2. run `npm install` to install dependencies.
3. run `npm run seed` to seed the data for setups.
4. run `npm run seedTag` to seed the data for tags.

# Tech Used: 
- Node.js
- Mongoose
- Mongodb
- Express
- Postman
